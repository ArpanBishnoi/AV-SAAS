 AV SaaS – AI Powered Notes Application

## Overview

AV SaaS is a secure AI-powered notes application built using FastAPI and Streamlit. Users can create an account, securely log in, manage their personal notes, and interact with an AI assistant.

The project demonstrates authentication, authorization, REST APIs, AI integration, deployment, and modern backend development practices.

---

## Features

* User Registration
* Secure Login System
* Password Hashing using bcrypt
* JWT Authentication
* Protected API Endpoints
* Create Notes
* Read Personal Notes
* Update Notes
* Delete Notes
* AI Question Answering
* Input Validation using Pydantic
* Streamlit Frontend
* FastAPI Backend
* Render Deployment

* JWT authentication is used for secure login sessions.
* Protected endpoints ensure users can access only their own data.
